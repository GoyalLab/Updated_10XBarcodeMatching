# FateMap 10X barcode extraction and matching pipeline
Purpose: Extract lineage barcodes (FateMap barcodes), cellIDs, and UMIs from 10X side reaction sequencing runs. The final output is a file that matches cellIDs to lineage barcodes that can be used to analyze 10X data.

Documentation can be found here: https://docs.google.com/document/d/1KCptW7_bAmO_9tRBW_N_VZQw4SCvsY7Erhjxuog8Rjs/edit?usp=sharing
